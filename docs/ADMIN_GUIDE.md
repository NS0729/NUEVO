# 商家后台使用指南

## 🚀 快速开始

### 访问后台

1. **方式一**: 点击导航栏右上角的"商家后台"图标（四个方块图标）
2. **方式二**: 直接访问 http://localhost:5173/admin/login

### 登录信息

**演示账户**:
- 用户名: `admin`
- 密码: `admin123`

## 📋 功能模块

### 1. 商品管理

**功能**:
- ✅ 查看所有商品列表
- ✅ 添加新商品
- ✅ 编辑商品信息
- ✅ 删除商品
- ✅ 查看商品详情

**操作步骤**:
1. 进入"商品管理"标签页
2. 点击"添加商品"按钮创建新商品
3. 点击"编辑"按钮修改商品信息
4. 点击"删除"按钮删除商品

### 2. 订单管理

**功能**:
- ✅ 查看所有订单
- ✅ 按状态筛选订单
- ✅ 更新订单状态
- ✅ 查看订单详情

**订单状态**:
- 待处理 (pending)
- 已确认 (confirmed)
- 已发货 (shipped)
- 已完成 (completed)
- 已取消 (cancelled)

### 3. 分类管理

**功能**:
- ✅ 查看所有分类
- ✅ 添加新分类
- ✅ 编辑分类信息
- ✅ 删除分类

## 🎨 界面说明

### 统计卡片
- 📦 商品总数
- 🛒 订单总数
- 💰 总销售额
- ⏳ 待处理订单

### 导航标签
- **商品管理**: 管理商品信息
- **订单管理**: 管理订单和状态
- **分类管理**: 管理商品分类

## 🔐 安全说明

- 当前使用简单的 localStorage 验证
- 生产环境应使用真实的身份验证系统
- 建议添加 API 权限验证

## 📝 注意事项

1. **数据同步**: 修改会保存到数据库
2. **图片URL**: 添加商品时需要提供有效的图片URL
3. **价格格式**: 价格以美元为单位，不需要输入符号
4. **必填字段**: 标有 * 的字段为必填项

## 🛠️ 技术说明

- **前端框架**: Vue 3
- **路由**: Vue Router
- **状态管理**: Pinia
- **后端API**: Cloudflare Workers + D1
- **样式**: SCSS

## 📍 访问地址

- **登录页面**: `/admin/login`
- **后台首页**: `/admin/dashboard`
- **商品管理**: `/admin/dashboard` (商品管理标签)
- **订单管理**: `/admin/dashboard` (订单管理标签)
- **分类管理**: `/admin/dashboard` (分类管理标签)

## ✨ 开始使用

1. 访问后台登录页面
2. 使用演示账户登录
3. 开始管理您的商品和订单！
