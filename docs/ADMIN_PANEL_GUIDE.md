# 商家后台管理系统使用指南

## 🎯 功能概述

商家后台管理系统是一个完整的商品、订单和分类管理平台，提供以下功能：

### ✨ 主要功能

1. **商品管理** 📦
   - 添加新商品
   - 编辑商品信息
   - 删除商品
   - 查看商品列表
   - 管理商品库存和精选状态

2. **订单管理** 🛒
   - 查看所有订单
   - 订单状态管理
   - 订单详情查看
   - 订单统计

3. **分类管理** 📁
   - 添加分类
   - 编辑分类
   - 删除分类
   - 分类列表管理

4. **数据统计** 📊
   - 商品总数
   - 订单总数
   - 总销售额
   - 待处理订单

## 🚀 快速开始

### 1. 启动服务

```bash
# 启动前端和后端
npm run dev

# 或分别启动
npm run dev:frontend   # 前端 (http://localhost:3000)
npm run dev:backend    # 后端 (http://localhost:8787)
```

### 2. 访问商家后台

1. 打开浏览器访问: `http://localhost:3000/admin/login`
2. 使用默认账号登录:
   - **用户名**: `admin`
   - **密码**: `admin123`

### 3. 开始管理

登录成功后，你将看到商家后台主界面，包含：
- 数据统计卡片
- 商品管理标签页
- 订单管理标签页
- 分类管理标签页

## 📋 详细功能说明

### 商品管理

#### 添加商品
1. 点击"商品管理"标签页
2. 点击"添加商品"按钮
3. 填写商品信息：
   - 商品名称（必填）
   - 分类（必填）
   - 价格（必填）
   - 原价（可选）
   - 商品图片（本地文件上传）
   - 商品描述
   - 材质、宝石、尺寸等信息
   - 库存状态
   - 是否精选
4. 点击"保存"按钮

#### 编辑商品
1. 在商品列表中，点击商品行的"编辑"按钮
2. 修改商品信息
3. 点击"保存"按钮

#### 删除商品
1. 在商品列表中，点击商品行的"删除"按钮
2. 确认删除操作

### 订单管理

#### 查看订单
1. 点击"订单管理"标签页
2. 查看所有订单列表
3. 点击订单查看详细信息

#### 更新订单状态
1. 在订单列表中找到目标订单
2. 点击"编辑"按钮
3. 修改订单状态
4. 保存更改

### 分类管理

#### 添加分类
1. 点击"分类管理"标签页
2. 点击"添加分类"按钮
3. 填写分类信息：
   - 分类ID（英文，如：rings）
   - 分类名称（中文，如：戒指）
   - 分类图标（emoji，如：💍）
4. 点击"保存"按钮

#### 编辑分类
1. 在分类列表中，点击分类的"编辑"按钮
2. 修改分类信息
3. 点击"保存"按钮

#### 删除分类
1. 在分类列表中，点击分类的"删除"按钮
2. 确认删除操作

## 🐛 常见问题

### 1. 无法登录
- 检查用户名和密码是否正确
- 检查后端服务是否运行
- 检查浏览器控制台是否有错误

### 2. 商品保存失败
- 检查必填字段是否都已填写
- 检查图片是否上传成功
- 检查后端 API 是否正常

### 3. 数据不显示
- 点击刷新按钮
- 检查网络连接
- 检查后端服务状态

### 4. 会话过期
- 重新登录
- 检查会话时间设置
- 在会话过期前保存工作
