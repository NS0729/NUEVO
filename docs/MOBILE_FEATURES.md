# 📱 移动端功能说明

## 已实现的移动端功能

### 1. ✅ 移动端底部导航栏
- **位置**: 固定在屏幕底部
- **功能**: 快速导航到首页、搜索、购物车
- **特性**:
  - 自动显示/隐藏（仅在移动端显示）
  - 购物车数量徽章
  - 活动状态高亮
  - 安全区域适配（iPhone X 等）

### 2. ✅ 触摸交互优化
- **触摸反馈**: 所有按钮都有触摸反馈动画
- **最小触摸目标**: 按钮最小高度 48px（符合移动端标准）
- **防止误触**: 优化触摸区域和间距
- **双击缩放**: 已禁用，防止意外缩放

### 3. ✅ 响应式布局
- **断点**: 
  - 移动端: ≤768px
  - 小屏移动端: ≤480px
- **自适应**: 所有组件都针对移动端优化
- **间距优化**: 移动端使用更紧凑的间距

### 4. ✅ 图片优化
- **懒加载**: 图片使用 `loading="lazy"` 属性
- **响应式图片**: 根据屏幕尺寸加载合适尺寸
- **加载动画**: 图片加载时的平滑过渡

### 5. ✅ 购物车优化
- **全屏模式**: 移动端购物车全屏显示
- **大按钮**: 数量控制按钮更大，易于点击
- **滑动操作**: 支持滑动删除（可扩展）

### 6. ✅ 手势支持工具
- **滑动检测**: 支持左滑、右滑、上滑、下滑
- **长按检测**: 支持长按操作
- **双击检测**: 支持双击操作

## 使用方法

### 移动端底部导航栏
自动在移动端（≤768px）显示，无需额外配置。

### 手势支持
```javascript
import { createSwipeDetector, createLongPressDetector } from '@/utils/touchGestures'

// 滑动检测
const cleanup = createSwipeDetector(element, {
  onSwipeLeft: () => console.log('左滑'),
  onSwipeRight: () => console.log('右滑'),
  onSwipeUp: () => console.log('上滑'),
  onSwipeDown: () => console.log('下滑')
})

// 清理
cleanup()
```

## 移动端优化细节

### 字体大小
- 移动端输入框字体 ≥16px（防止 iOS 自动缩放）
- 标题字体自适应缩小

### 触摸目标
- 所有可点击元素最小 44×44px
- 按钮最小高度 48px

### 性能优化
- 图片懒加载
- 触摸事件使用 `passive: true`
- CSS 动画使用 `transform` 和 `opacity`

### 安全区域
- 底部导航栏适配 iPhone X 等设备的安全区域
- 使用 `env(safe-area-inset-bottom)` 添加底部内边距
